# 操作とか
__起動したら最初にアムレンとスプリントのキー設定をしてください__<br>
前移動：W<br>
後移動：S<br>
左移動：A<br>
右移動：D<br>
リトライ：Y<br>
カメラ移動：マウス操作<br>
カメラ固定：SPACE長押し<br>
戻る：ESC<br>
アムレン：おこのみで
スプリント：おこのみで

# 既知のバグ
### NPCが死ぬ
直線移動しかできないので結構死にます。<br>
気になるようだったら直します・・・
